References

These resources were used to research HTML, CSS, JavaScript, responsive design, and modern web development best practices.

  1. MDN Web Docs — Used for reliable explanations of HTML, CSS, JavaScript, and browser standards.
  2. W3Schools — Used for beginner-friendly examples of HTML tags, CSS properties, and JavaScript basics.
  3. HTML Living Standard — Used to understand official HTML structure and webpage elements.
  4. W3C CSS — Used to research CSS standards and styling practices.
  5. MDN Responsive Design — Used to understand responsive layouts, flexible design, and media queries.
  6. web.dev Responsive Design — Used to study mobile-friendly design and modern layout practices.
  7. MDN Media Queries — Used to learn how CSS media queries adjust designs for different screen sizes.
  8. GitHub Pages — Used to understand how to deploy and publish a static website.
  9. MDN JavaScript — Used to build the quiz behavior with form inputs, buttons, and results.
  10. Nielsen Norman Group Responsive Design — Used to understand usability and responsive design best practices.
