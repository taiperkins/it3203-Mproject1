HTML

HTML stands for HyperText Markup Language. It creates the structure of a webpage by using elements such as headings, paragraphs, links, images, forms, and tables.

CSS

CSS stands for Cascading Style Sheets. It controls how webpages look, including colors, fonts, spacing, layout, borders, and responsive design.

JavaScript

JavaScript adds behavior to webpages. It can validate forms, respond to button clicks, update page content, and create interactive features such as quizzes.

Responsive Design

Responsive design makes websites adjust to different screen sizes. This project uses flexible layouts, responsive images, and media queries to support desktop, tablet, and phone screens.

Frontend and Backend Development

Frontend development focuses on what users see in the browser. Backend development handles servers, databases, and application logic behind the scenes.
